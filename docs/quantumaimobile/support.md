# QuantumAIMobile Support

Last updated: March 27, 2026

QuantumAIMobile provides market monitoring, strategy simulation, wallet activation flows, and operational controls in a single iOS app.

## Support contact

Email: erenuludemir@icloud.com

## When contacting support

Please include:

- device model
- iOS version
- app version and build number
- a short description of the issue
- screenshots or screen recordings when relevant

## Common topics

- market data or bridge refresh issues
- wallet activation and verification flow issues
- strategy preset behavior
- app performance or memory usage
- App Store review access questions

## Response scope

Support is provided for the published iOS app and its core features.
