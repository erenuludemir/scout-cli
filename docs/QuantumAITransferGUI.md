# QuantumAITransferGUI.app Kullanım Kılavuzu

## Tanım
SwiftUI tabanlı GUI uygulamasıdır. Menü çubuğundan manuel ya da otomatik ETH/TRON işlemleri başlatır.

## Çalıştırma
```
open /QuantumAI-Transfer-System/QuantumAITransferGUI.app
```

## Notlar
- Tüm .app dosyaları macOS için optimize edilmiştir.
- launchd veya Docker içinde otomatik başlatılabilir.
